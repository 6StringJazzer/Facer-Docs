--- title: Backgrounds category: Element Types order: 1 ---

Background elements are simple elements that are always included in your watch face, and stay at the bottom of your layers.

Background Image

The background image is similar to the custom 'image' element, except that it is always in your layer stack. This element allows advanced settings and tags to allow for dynamic backgrounds.

Background Color

The background color is the default color that will be shown to a user if there are no other layers on top.
