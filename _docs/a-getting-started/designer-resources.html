Watch face design

Good watch face design typically has a few characteristics, just like app design. They are focused, have a clear hierarchy and have great color harmony. Keep all of those in mind when designing.

Design specs

Canvas is 640 x 640px at 72 dpi/ppi. You might also want to consider that some watches have a chin at the bottom which would hide any design elements that would be down there on selected models. The editor also gives an option to preview the chin with the leftmost button, which is located right below canvas preview

Make sure you design for both square and circle.

You can do this by clicking advanced button on an image layer and then uploading different images for square and circle.

Best Practices

Crisp Design

When designing in Photoshop/Illustrator or other programs, make sure you are always locking to pixels and on even numbers. Also make sure the minimum pixel stroke is 2px.

Good performance

Make sure all your layers are compressed into a single layer if they are part of the background. The less layers you have the quicker it will sync and smoother it will run.

Keep it on brand

When adding a description make sure you are selling the watch and not the features. We handle most of the features in the user interface so you should stay focused on why you made the watch and why the user will want to wear it. For example "This is a watchface any SCI-FI lover will like. Feel like you are in the driver's seat of a spaceship"
